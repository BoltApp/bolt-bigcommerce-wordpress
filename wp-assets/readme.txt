== Installation ==
In addition to BigCommerce account, a Bolt merchant account is required for setup. Please contact [sales@bolt.com](mailto:sales@bolt.com) or schedule a check-in with our team at [bolt.com](bolt.com) to check your eligibility.

= Automatic Installation =
Automatic installation is the easiest option as WordPress handles the file transfers. Log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

= Manual Installation =
The manual installation method involves downloading our plugin and uploading it to your server via SFTP. Please contact Bolt team for further instruction.
